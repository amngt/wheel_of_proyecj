# Vite Project Base

## Dev Dependencies
- Vite [https://vitejs.dev/]
- Sass [https://sass-lang.com/]
- Typescript [https://www.typescriptlang.org/]

## Installation
```
npm install
npm run dev
```

## Production
```
npm run build
npm run preview
```
